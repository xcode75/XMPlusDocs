CF Zone

Zones

Adding a Cloudflare zone can help you add, update, and delete Cloudflare DNS of nodes without logging in to Cloudflare to resolve DNS every time.

Login to Cloudflare

Website -> Home -> Choose your domain name -> API Get Zone ID

API Token -> API Key -> Global API Key Get API Token

Add Zone

Require

☑️ CF mail ☑️ CF domain name ☑️ CF Zone ID ☑️ CF API key

XMPlus Panel -> Zone List -> Add Zone
